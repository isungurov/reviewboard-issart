==========
Screenshot
==========

Fields
======

+----------------+-------------+-------------------------------------------+
| Field          | Type        | Description                               |
+================+=============+===========================================+
| id             | Integer     | The ID number of the screenshot.          |
+----------------+-------------+-------------------------------------------+
| caption        | String      | The public caption on the screenshot.     |
+----------------+-------------+-------------------------------------------+
| title          | String      | Title text for the screenshot. This       |
|                |             | differs from the caption in that it will  |
|                |             | always be in the form of "Screenshot:     |
|                |             | <caption-or-filename>"                    |
+----------------+-------------+-------------------------------------------+
| image_url      | String      | The URL to the image.                     |
+----------------+-------------+-------------------------------------------+
| thumbnail_url  | String      | The URL to the thumbnail.                 |
+----------------+-------------+-------------------------------------------+
